readqr.py
----------------------------
def read()
カメラを起動してQRコードを読み取り、QRコードに埋め込まれてる本の名前を返す
引数は不要


def get_booksvalue()
csv形式で保存されたデータを読み取って現在何冊存在するのかを返す
引数は不要


def load_data()
csv形式で保存されたデータを読み取ってリスト型にして返す
データの構造は
data=["名前",0 or 1,"借りている人の名前"]
引数は不要


def check()
上3つの関数をまとめてその本が借りれるかどうかを表示してくれるやつ
